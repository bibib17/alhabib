Pengenalan CSS

Apa itu CSS, cara kerjanya, dan bagaimana cara menulisnya

💡

Apa itu CSS?

CSS (Cascading Style Sheets) adalah bahasa yang digunakan untuk menentukan tampilan dan gaya halaman web yang ditulis dalam HTML. Jika HTML adalah "kerangka" website, maka CSS adalah "pakaian" dan "dekorasinya".

📝

Analogi mudah: HTML = bangunan rumah, CSS = cat dinding, furnitur, dekorasi.

CSS memungkinkan kita mengontrol:

  • Warna teks dan latar belakang
  • Ukuran dan jenis font
  • Tata letak dan posisi elemen
  • Efek visual seperti bayangan dan gradasi
  • Animasi dan transisi
🔗

3 Cara Menerapkan CSS

External
Internal
Inline

File CSS terpisah, dihubungkan lewat tag <link>. Cara terbaik dan paling umum digunakan.

HTML
<!-- Di dalam <head> HTML -->
<link rel="stylesheet" href="style.css">

CSS ditulis di dalam tag <style> di bagian <head> HTML.

HTML
<style>
  p {
    color: blue;
  }
</style>

CSS ditulis langsung di atribut style elemen HTML. Tidak disarankan untuk proyek besar.

HTML
<p style="color: blue; font-size: 18px;">
  Teks berwarna biru
</p>
⚙️

Sintaks CSS

Aturan CSS terdiri dari selektor dan deklarasi. Deklarasi berisi properti dan nilai.

CSS
/* Struktur dasar CSS */
selektor {
    properti: nilai;
    properti2: nilai2;
}

/* Contoh nyata */
h1 {
    color: #e85d2f;
    font-size: 2rem;
    font-weight: bold;
}
💡

Setiap deklarasi diakhiri dengan titik koma (;). Lupa tanda ini adalah kesalahan paling umum!

⬇️

Apa itu "Cascading"?

Kata "Cascading" berarti "mengalir ke bawah". CSS mengikuti aturan:

  • Urutan: CSS yang ditulis belakangan akan menimpa yang sebelumnya.
  • Spesifisitas: Selektor yang lebih spesifik memiliki prioritas lebih tinggi.
  • Warisan (Inheritance): Beberapa properti diwarisi dari elemen induk ke anak.
CSS
p { color: blue; }   /* aturan 1 */
p { color: red;  }   /* aturan 2 menang! */
/* Teks paragraf akan berwarna merah */

Selektor CSS

Cara memilih elemen HTML yang ingin diberi gaya

🔡

Jenis-Jenis Selektor Dasar

SelektorContohKeterangan

ID#header { }Elemen dengan id="header"

CSS
/* Selektor elemen */
p { color: gray; }

/* Selektor class (pakai titik) */
.teks-merah { color: red; }

/* Selektor ID (pakai #) */
#judul-utama { font-size: 2rem; }

/* Universal — mempengaruhi semua elemen */
* { box-sizing: border-box; }
🔗

Selektor Kombinasi

CSS
/* Descendant — semua p di dalam div */
div p { color: blue; }

/* Child — hanya p langsung di dalam div */
div > p { font-weight: bold; }

/* Adjacent sibling — p tepat setelah h2 */
h2 + p { font-style: italic; }

/* General sibling — semua p setelah h2 */
h2 ~ p { color: gray; }

/* Gabungan beberapa selektor */
h1, h2, h3 { font-family: Georgia, serif; }

Pseudo-class & Pseudo-element

Pseudo-class memilih elemen berdasarkan state/kondisi, pseudo-element memilih bagian tertentu dari elemen.

CSS
/* Pseudo-class — state elemen */
a:hover   { color: red; }
a:visited { color: purple; }
input:focus { border: 2px solid blue; }
li:first-child { font-weight: bold; }
li:nth-child(2n) { background: #f0f0f0; }

/* Pseudo-element — bagian elemen */
p::first-letter { font-size: 2em; }
p::before { content: "→ "; }
::selection { background: yellow; }

Box Model CSS

Setiap elemen HTML adalah sebuah kotak — pahami strukturnya

📦

Struktur Box Model

Setiap elemen HTML terdiri dari 4 lapisan box model dari dalam ke luar:

Visual
margin
border
padding
content
  • Content — Area konten (teks, gambar)
  • Padding — Jarak antara konten dan border (dalam)
  • Border — Garis tepi elemen
  • Margin — Jarak luar antar elemen
📏

Properti Box Model

CSS
.kotak {
  /* Ukuran */
  width: 300px;
  height: 200px;

  /* Padding (semua sisi) */
  padding: 20px;
  /* atau per sisi */
  padding: 10px 20px 10px 20px;
  /* atas kanan bawah kiri */

  /* Border */
  border: 2px solid #333;
  border-radius: 8px;

  /* Margin */
  margin: 16px auto;
  /* atas/bawah 16px, kiri/kanan auto (center) */
}
🔲

box-sizing: border-box

Secara default, width hanya menghitung area konten. Dengan box-sizing: border-box, padding dan border sudah termasuk dalam lebar yang ditentukan.

CSS
/* TANPA border-box */
.kotak {
  width: 200px;
  padding: 20px;
  /* Total lebar = 200 + 40 = 240px */
}

/* DENGAN border-box (direkomendasikan) */
* { box-sizing: border-box; }
.kotak {
  width: 200px;
  padding: 20px;
  /* Total lebar = tetap 200px */
}
💡

Hampir semua developer modern menggunakan * { box-sizing: border-box; } di setiap project.

Tipografi CSS

Mengatur tampilan teks agar nyaman dibaca dan menarik secara visual

🖋️

Properti Font Utama

CSS
body {
  /* Jenis font */
  font-family: 'Roboto', Arial, sans-serif;

  /* Ukuran font */
  font-size: 16px;   /* atau rem, em, % */

  /* Ketebalan */
  font-weight: 400;  /* 100-900 atau bold */

  letter-spacing: 0.5px;

  word-spacing: 2px;

CSS Variables untuk Warna

CSS Variables (custom properties) sangat berguna untuk menyimpan warna tema agar mudah diubah.

  --warna-sekunder: #2e7d8c;

  color: var(--warna-utama);
}

}

CSS Flexbox

Layout satu dimensi yang fleksibel — solusi terbaik untuk menyusun item secara baris atau kolom

🧭

Konsep Dasar Flexbox

Flexbox bekerja dengan container (parent) dan item (child). Aktifkan dengan display: flex.

CSS
.container {
  display: flex;

  /* Arah utama */
  flex-direction: row;       /* row|column|row-reverse */

  /* Penempatan di sumbu utama */
  justify-content: center;   /* flex-start|end|center|space-between|space-around */

  /* Penempatan di sumbu silang */
  align-items: center;      /* stretch|flex-start|flex-end|center|baseline */

  /* Jarak antar item */
  gap: 16px;

  /* Wrap item */
  flex-wrap: wrap;
}
🎮

Properti Flex Item

CSS
.item {
  /* Seberapa item bisa tumbuh */
  flex-grow: 1;   /* 0 = tidak tumbuh */

  /* Seberapa item bisa mengecil */
  flex-shrink: 1;

  /* Ukuran dasar item */
  flex-basis: 200px;

  /* Shorthand: grow shrink basis */
  flex: 1 1 200px;

  /* Urutan tampil */
  order: 2;

  /* Override align-items untuk 1 item */
  align-self: flex-end;
}
💡

Use Case Flexbox Populer

CSS
/* Pusatkan elemen secara sempurna */
.center {
  display: flex;
  justify-content: center;
  align-items: center;
}

/* Navbar dengan logo kiri, menu kanan */
.navbar {
  display: flex;
  justify-content: space-between;
  align-items: center;
}

/* Kartu produk yang fleksibel */
.cards {
  display: flex;
  flex-wrap: wrap;
  gap: 16px;
}
.card { flex: 1 1 250px; }

CSS Transition

Transition membuat perubahan properti CSS menjadi animasi yang halus.

CSS
.tombol {
  background: #e85d2f;
  /* property durasi timing-function delay */
  transition: all 0.3s ease;
  /* atau spesifik: */
  transition: background 0.3s ease,
    transform 0.2s ease;
}
.tombol:hover {
  background: #c44b20;
  transform: translateY(-2px);
}

CSS Animation & Keyframes

CSS
/* 1. Definisikan keyframes */
@keyframes bouncing {
  0%   { transform: translateY(0); }
  50%  { transform: translateY(-40px); }
  100% { transform: translateY(0); }
}

/* 2. Terapkan ke elemen */
.bola {
  animation-name: bouncing;
  animation-duration: 1s;
  animation-iteration-count: infinite;
  animation-timing-function: ease-in-out;
  /* shorthand: */
  animation: bouncing 1s infinite ease-in-out;
}

CSS Transform

CSS
.elemen {
  /* Pindah posisi */
  transform: translateX(20px) translateY(-10px);

  /* Putar */
  transform: rotate(45deg);

  /* Perbesar/perkecil */
  transform: scale(1.2);

  /* Memiringkan */
  transform: skewX(15deg);

  /* Gabungkan beberapa transform */
  transform: translateY(-4px) scale(1.05);
}
⚡

Transform dan opacity diproses oleh GPU — gunakan keduanya untuk animasi yang performa-nya tinggi!

Media Query

Media query mengaplikasikan CSS berdasarkan kondisi tertentu (lebar layar, orientasi, dll).

CSS
/* Mobile First — mulai dari layar kecil */
.kolom {
  grid-template-columns: 1fr; /* 1 kolom */
}

/* Tablet (≥768px) */
@media (min-width: 768px) {
  .kolom {
    grid-template-columns: 1fr 1fr;
  }
}

/* Desktop (≥1024px) */
@media (min-width: 1024px) {
  .kolom {
    grid-template-columns: 1fr 1fr 1fr;
  }
}

/* Orientasi landscape */
@media (orientation: landscape) {
  .hero { height: 100vh; }
}

Viewport Meta & Fluid Layout

HTML
<!-- Wajib ada di <head> -->
<meta name="viewport"
  content="width=device-width, initial-scale=1.0">
CSS
/* Gambar responsif */
img {
  max-width: 100%;
  height: auto;
}

/* Font responsif dengan clamp */
h1 {
  font-size: clamp(1.5rem, 5vw, 3rem);
  /* min, preferred, max */
}

/* Kontainer responsif */
.container {
  width: min(1200px, 100% - 32px);
  margin-inline: auto;
}

/* Grid responsif tanpa media query! */
.auto-grid {
  display: grid;
  grid-template-columns:
    repeat(auto-fit, minmax(250px, 1fr));
}
