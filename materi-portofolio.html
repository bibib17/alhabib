🧱 HTML

Bagian <head> — Setup Halaman

Bagian <head> berisi "pengaturan tersembunyi" yang tidak tampil di layar, tapi sangat penting untuk meta informasi, stylesheet, dan title halaman.

index.html — bagian <head> 
<head>
  <!-- 1. Encoding karakter —— wajib ada! -->
  <meta charset="utf-8" />

  <!-- 2. Agar tampilan responsif di HP -->
  <meta name="viewport" content="width=device-width, initial-scale=1" />

  <!-- 3. Judul tab browser -->
  <title>My Portfolio | Haikal Faturrahman Faizera</title>

  <!-- 4. Bootstrap CSS dari CDN -->
  <link href="https://cdn.jsdelivr.net/npm/bootstrap@5.3.8/dist/css/bootstrap.min.css"
        rel="stylesheet" />

  <!-- 5. Bootstrap Icons (library ikon) -->
  <link rel="stylesheet"
        href="https://cdn.jsdelivr.net/npm/bootstrap-icons@1.11.1/font/bootstrap-icons.css" />

  <!-- 6. CSS kustom kita sendiri -->
  <link rel="stylesheet" href="style.css" />
</head>
1

charset="utf-8"

Memberitahu browser bahwa halaman ini menggunakan UTF-8 — format encoding yang mendukung hampir semua bahasa, termasuk huruf Indonesia & emoji. Tanpa ini, karakter bisa kacau.

2

viewport meta

width=device-width artinya lebar halaman sama dengan lebar layar perangkat. initial-scale=1 artinya tidak ada zoom awal. Ini wajib agar Bootstrap responsive bekerja.

3

Bootstrap + Icons dari CDN

Bootstrap dimuat dari cdn.jsdelivr.net — server CDN yang sangat cepat. Perhatikan urutan: Bootstrap dulu, baru style.css kita. Ini penting agar CSS kita bisa "menimpa" style Bootstrap.

⚠️ Urutan Link CSS Sangat Penting!
style.css kita harus ditempatkan setelah Bootstrap. Jika dibalik, style kustom kita akan tertimpa Bootstrap dan tidak berpengaruh.

🔍 Bedah Class Bootstrap pada Navbar

✅
Cara Kerja Hamburger Menu data-bs-toggle="collapse" dan data-bs-target="#navbarNav" menghubungkan tombol dengan area menu. Saat tombol diklik, Bootstrap secara otomatis menampilkan/menyembunyikan menu. Ini tanpa JavaScript tambahan!

🖼 Class pada Gambar Profil

  • rounded-circle — membuat gambar berbentuk lingkaran penuh
  • img-thumbnail — menambahkan border putih dan padding di sekeliling gambar
  • width="200" — ukuran 200px (bisa juga pakai CSS)

✍️ Class Typography

  • display-4 — heading sangat besar (Bootstrap display heading)
  • lead — teks intro yang sedikit lebih besar dari normal
  • text-center — rata tengah semua konten di dalam section
🌊
Apa itu SVG Wave? SVG (Scalable Vector Graphics) adalah format gambar berbasis kode yang bisa di-scale tanpa pecah. Wave di sini berfungsi sebagai transisi visual antara section berwarna dan section putih. Warna fill="#ffffff" harus sesuai dengan warna background section berikutnya.

    <label for="email" class="form-label">Email</label>
    <input type="email"
           class="form-control"
           id="email" />
  </div>

  <!-- Textarea Pesan -->
  <div class="mb-3">
    <label for="pesan" class="form-label">Pesan</label>
    <textarea class="form-control"
              id="pesan" rows="3"></textarea>
  </div>

  <!-- Tombol Submit -->
  <button type="submit"
          class="btn btn-primary">Kirim</button>
</form>

📋 Class Form Bootstrap

  • form-label — style label yang rapi di atas input
  • form-control — style input/textarea: border, padding, focus ring
  • mb-3 — margin bawah untuk jarak antar field
  • btn btn-primary — tombol biru standar Bootstrap

🔗 Pentingnya for & id

Attribute for pada label harus sama dengan id pada input. Ini membuat klik pada label langsung fokus ke input-nya — meningkatkan aksesibilitas dan UX.

🎨 CSS

style.css — Kustomisasi Bootstrap

File style.css berisi CSS kustom yang "menimpa" (override) style default Bootstrap untuk menyesuaikan tampilan portfolio.

style.css — isi lengkap 
/* Jumbotron — Hero Section */
.jumbotron {
  padding-top: 6rem;         /* beri ruang untuk navbar fixed */
  background-color: #e2edff; /* biru muda cerah */
}

/* Project Section */
#project {
  background-color: #e2edff; /* sama dengan jumbotron */
}

/* Semua section — padding atas */
section {
  padding-top: 6rem; /* agar konten tidak tertutup navbar */
}

🤔 Mengapa padding-top: 6rem?

Navbar menggunakan fixed-top, artinya navbar "mengambang" di atas konten. Tanpa padding-top, konten pertama setiap section akan tertutup di balik navbar. Padding 6rem (≈96px) memberi ruang agar konten tidak tersembunyi.

❌ Tanpa padding-top
section {
  /* tidak ada padding */
}
/* Akibat: judul section
   tersembunyi di balik navbar! */
✅ Dengan padding-top
section {
  padding-top: 6rem;
}
/* Akibat: konten terlihat
   jelas di bawah navbar */
🎯
CSS Override — Cara Bootstrap Dikustomisasi Bootstrap adalah framework — ia punya style defaultnya sendiri. Kita bisa menimpa style tersebut dengan membuat CSS kustom. Karena style.css dimuat setelah Bootstrap di <head>, rule kita akan menang (spesifisitas sama, yang belakangan yang berlaku).
💡 Kembangkan style.css kamu!
Kamu bisa menambahkan: animasi hover pada card, font Google Fonts, warna brand sendiri, efek scroll, dan lain-lain. Style CSS kustom inilah yang membuat portfoliomu unik dan berbeda dari yang lain.

Di project ini dipakai col-md-4: artinya 4 kolom pada layar ≥768px. Di bawah 768px → otomatis jadi col-12 (full width, 1 kolom per baris).

✅ Struktur Wajib Grid:
.container → .row → .col-*
Jangan pernah skip satu level! Row harus di dalam container, dan col harus langsung di dalam row.
