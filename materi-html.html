Pengenalan HTML

Apa itu HTML, sejarahnya, dan bagaimana cara kerjanya

Apa itu HTML?

HTML (HyperText Markup Language) adalah bahasa standar untuk membuat dan menyusun konten di web. HTML bukan bahasa pemrograman — ia adalah bahasa markup yang menggunakan tag untuk mendeskripsikan struktur halaman web.

💡

Setiap halaman web yang kamu buka di browser dibuat dengan HTML. HTML mendefinisikan struktur konten, CSS mengatur tampilan, dan JavaScript menambahkan interaktivitas.

Sejarah Singkat HTML

TahunVersiKeterangan
1991HTML 1.0Diciptakan Tim Berners-Lee di CERN
1995HTML 2.0Standar pertama oleh IETF
1997HTML 3.2 & 4.0Dukungan CSS & JavaScript
2000XHTML 1.0HTML berbasis XML yang lebih ketat
2014HTML5Standar modern dengan fitur multimedia & semantik
LivingHTML Living StandardDikelola WHATWG, terus berkembang

Anatomi Tag HTML

Elemen HTML terdiri dari opening tag, konten, dan closing tag:

Anatomi Elemen HTML 
<!-- Struktur dasar elemen HTML -->

<p class="intro">Ini adalah paragraf.</p>
 ↑                    ↑             ↑
 Opening tag          Konten        Closing tag

<!-- Elemen self-closing (tidak butuh closing tag) -->
<br>      <!-- line break -->
<img src="foto.jpg">
<input type="text">
⚡

Tips: Dalam HTML5, tag self-closing tidak perlu slash di akhir (<br> bukan <br />), meski keduanya valid.

Cara Kerja Browser

Ketika kamu membuka halaman web, browser melakukan proses berikut:

  1. Mengunduh file HTML dari server
  2. Mem-parsing HTML menjadi DOM (Document Object Model)
  3. Mengunduh CSS dan menerapkan gaya
  4. Menjalankan JavaScript
  5. Merender halaman yang sudah jadi ke layar

Struktur Dokumen HTML

Kerangka dasar yang wajib ada di setiap halaman HTML

Kerangka Dasar HTML5

Setiap dokumen HTML yang valid harus memiliki struktur berikut:

index.html 
<!DOCTYPE html>
<html lang="id">
<head>
  <meta charset="UTF-8">
  <meta name="viewport" content="width=device-width, initial-scale=1.0">
  <title>Judul Halaman Web</title>
</head>
<body>
  <h1>Halo Dunia!</h1>
  <p>Ini adalah halaman HTML pertama saya.</p>
</body>
</html>

Penjelasan Setiap Bagian

TagFungsi
<!DOCTYPE html>Deklarasi tipe dokumen — memberitahu browser ini adalah HTML5
<html>Elemen root (akar) dari seluruh halaman HTML
lang="id"Atribut bahasa — penting untuk aksesibilitas & SEO
<head>Berisi metadata — tidak ditampilkan langsung di halaman
<meta charset>Menentukan encoding karakter (UTF-8 mendukung semua karakter)
<meta viewport>Membuat halaman responsif di perangkat mobile
<title>Judul yang muncul di tab browser dan hasil pencarian
<body>Semua konten yang ditampilkan di halaman berada di sini

Isi Tag <head>

Tag head bisa berisi berbagai elemen penting:

Contoh lengkap <head> 
<head>
  <!-- Encoding karakter -->
  <meta charset="UTF-8">

  <!-- Responsif di mobile -->
  <meta name="viewport" content="width=device-width, initial-scale=1.0">

  <!-- Judul halaman -->
  <title>Nama Website Saya</title>

  <!-- Deskripsi untuk SEO -->
  <meta name="description" content="Deskripsi halaman ini">

  <!-- Link ke file CSS -->
  <link rel="stylesheet" href="style.css">

  <!-- CSS langsung (internal) -->
  <style>
    body { font-family: Arial; }
  </style>
</head>

Teks & Heading

Tag Heading (H1-H6)

HTML memiliki 6 level heading. H1 adalah yang terbesar dan paling penting — digunakan sebagai judul utama halaman. H2-H6 untuk sub-judul dan sub-sub-judul.

Heading H1 - H6

<h2>Heading 2 — Sub Judul</h2>
<h3>Heading 3 — Sub Sub Judul</h3>
<h4>Heading 4</h4>
<h5>Heading 5</h5>
<h6>Heading 6 — Paling kecil</h6>

Link & Gambar

Navigasi antar halaman dan menampilkan gambar

Tag Anchor — Membuat Link

Tag <a> digunakan untuk membuat hyperlink ke halaman lain, file, atau bagian dalam halaman.

Berbagai jenis link 
<!-- Link ke website eksternal -->
<a href="https://www.google.com" target="_blank">Buka Google</a>

<!-- Link ke halaman lain (relatif) -->
<a href="tentang.html">Tentang Kami</a>
<a href="folder/kontak.html">Kontak</a>

<!-- Link ke bagian di halaman yang sama (anchor) -->
<a href="#section-2">Lompat ke Bagian 2</a>
<h2 id="section-2">Ini Bagian 2</h2>

<!-- Link email -->
<a href="mailto:email@contoh.com">Kirim Email</a>

<!-- Link telepon -->
<a href="tel:+6281234567890">Hubungi Kami</a>

<!-- Link unduh file -->
<a href="dokumen.pdf" download>Unduh PDF</a>
hrefURL tujuan link
target="_blank"Buka di tab baru
target="_self"Buka di tab yang sama (default)
downloadUnduh file daripada buka
titleTooltip saat hover
rel="noopener"Keamanan untuk _blank

Tag Gambar — <img>

Menampilkan gambar 
<!-- Gambar dari file lokal -->
<img src="gambar/foto.jpg" alt="Deskripsi gambar">

<!-- Gambar dari internet -->
<img 
  src="https://picsum.photos/400/200"
  alt="Foto acak"
  width="400"
  height="200"
  loading="lazy"
>

<!-- Gambar sebagai link -->
<a href="halaman.html">
  <img src="logo.png" alt="Logo">
</a>

<!-- Figure dengan caption -->
<figure>
  <img src="gunung.jpg" alt="Pemandangan gunung">
  <figcaption>Pemandangan Gunung Bromo saat fajar</figcaption>
</figure>
⚠️

Atribut alt sangat penting! Alt text mendeskripsikan gambar untuk screen reader (aksesibilitas) dan ditampilkan jika gambar gagal dimuat. Selalu gunakan alt text yang deskriptif.

  <li>HTML</li>

  <li>JavaScript</li>

  <li>Buka editor teks</li>

  <li>Backend</li>

<!-- Definition List -->

  <dt>HTML</dt>
  <dd>HyperText Markup Language</dd>
  <dt>CSS</dt>
  <dd>Cascading Style Sheets</dd>

Div, Span & HTML Semantik

Struktur layout dan elemen semantik HTML5

Div vs Span

<div> adalah elemen block-level — selalu mulai di baris baru. <span> adalah elemen inline — mengalir dalam teks.

Div dan Span 
<!-- DIV: container block-level -->
<div class="container" id="main">
  <div class="header">Header</div>
  <div class="content">
    <p>Paragraf biasa dengan <span style="color:red">teks merah</span> di dalamnya.</p>
  </div>
  <div class="footer">Footer</div>
</div>

<!-- SPAN: inline container -->
<p>Harga: <span class="harga">Rp 50.000</span> per item</p>

HTML5 Semantic Elements

Elemen semantik memberikan makna pada konten — membantu browser, mesin pencari, dan screen reader memahami struktur halaman.

Layout semantik HTML5 
<body>
  <!-- Header halaman -->
  <header>
    <nav>
      <a href="/">Beranda</a>
      <a href="/tentang">Tentang</a>
    </nav>
  </header>

  <!-- Konten utama -->
  <main>
    <article>
      <header>
        <h1>Judul Artikel</h1>
        <time datetime="2025-01-15">15 Januari 2025</time>
      </header>
      <section>
        <h2>Bagian 1</h2>
        <p>Isi artikel...</p>
      </section>
    </article>

    <!-- Konten sampingan -->
    <aside>
      <h3>Artikel Terkait</h3>
    </aside>
  </main>

  <!-- Footer -->
  <footer>
    <p>&copy; 2025 Website Saya</p>
  </footer>
</body>
Tag SemantikKegunaan
<header>Header halaman atau section
<nav>Navigasi utama
<main>Konten utama (hanya 1 per halaman)
<article>Konten mandiri (artikel, post blog)
<section>Bagian tematik dari konten
<aside>Konten sampingan / sidebar
<footer>Footer halaman atau section
<figure>Gambar, diagram, atau illustrasi
<figcaption>Caption untuk figure
<time>Tanggal atau waktu
<address>Informasi kontak
<details> + <summary>Accordion/dropdown konten

  ctx.fillStyle = 'blue';
  ctx.fillText('Hello Canvas!', 150, 60);
</script>

Meta, Head & Atribut Global

SEO, aksesibilitas, dan atribut yang berlaku di semua elemen

Tag Meta Penting

Meta tags lengkap 
<head>
  <!-- Wajib ada -->
  <meta charset="UTF-8">
  <meta name="viewport" content="width=device-width, initial-scale=1.0">
  <title>Judul Halaman — Nama Website</title>

  <!-- SEO -->
  <meta name="description" content="Deskripsi singkat halaman ini untuk Google (150-160 karakter)">
  <meta name="keywords" content="html, belajar, tutorial, web">
  <meta name="author" content="Nama Kamu">
  <meta name="robots" content="index, follow">

  <!-- Open Graph (untuk share di sosmed) -->
  <meta property="og:title" content="Judul untuk Share">
  <meta property="og:description" content="Deskripsi saat dibagikan">
  <meta property="og:image" content="https://contoh.com/gambar.jpg">

  <!-- Favicon -->
  <link rel="icon" href="favicon.ico">
  <link rel="icon" type="image/png" href="favicon.png">

  <!-- Stylesheet -->
  <link rel="stylesheet" href="style.css">

  <!-- Refresh otomatis setiap 30 detik -->
  <meta http-equiv="refresh" content="30">
</head>
